🗾 日本旅行思い出マップ

都道府県ごとの旅行記録を保存し、訪問状況を視覚的に管理できるWebアプリケーションです。
📜 概要

本アプリは、訪れた都道府県の思い出（コメントや写真URL）を記録するためのツールです。 バックエンドに Supabase を採用することで、簡易的なデータベース（SQLite3等）では難しかったデータの永続化を実現しています。アプリを停止・再起動しても、大切な旅行の記録が消えることはありません。

✨ 特徴

    ビジュアル進捗管理: 47都道府県のタイルマップにより、制覇状況が直感的に把握できます。

    データの永続保存: Supabaseとの連携により、クラウド上に安全にデータが保管されます。

    直感的な操作: マップ上のボタンをクリックするだけで、その県の思い出を瞬時に絞り込めます。

    レスポンシブデザイン: PCだけでなく、スマートフォンからの閲覧・入力にも対応しています。

🚀 使い方

    旅行を記録する

        サイドバーのフォームから「都道府県」「日付」「思い出」「写真URL（任意）」を入力し、「保存」をクリックします。

    訪問状況を見る

        メイン画面のタイルマップを確認します。訪問済みの県には自動的に「✅」マークが付きます。

    思い出を振り返る

        タイルマップの県名ボタンを押すと、右側のフィードにその県の過去の記録が表示されます。

🛠 実行環境・設定
    Python 3.11+
